## 启动Spring boot

#### 安装node.js

```
dnf install nodejs
```

#### 安装pm2

```
npm install pm2 -g
```

#### 编写启动文件 pm2.json

```json
{
  "name": "api",
  "script": "/usr/bin/java",
  "args": ["-jar", "PatientCommunication-0.0.1-SNAPSHOT.jar"],
  "exec_interpreter": "",
  "exec_mode": "fork"
}
```

#### 启动

```
pm2 start pm2.json
```
